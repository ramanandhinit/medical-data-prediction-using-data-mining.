# medical-data-prediction-using-data-mining
Medical Data Prediction Using Data Mining is a healthcare system that predicts diabetes using machine learning techniques. It analyzes patient data such as glucose, BMI, blood pressure, insulin, and age to provide accurate results. The system helps in early detection, prevention, and better health decision-making.
# Medical Data Prediction using Data Mining

This project predicts the possibility of diabetes using data mining techniques.

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- WAMP Server

## Description
The system allows users to input health parameters such as glucose level, BMI, age, blood pressure, etc.
Based on trained datasets (Pima Indian Diabetes Dataset), the system predicts the likelihood of diabetes.

## Features
- User registration & login
- Diabetes prediction
- Admin & Doctor modules
- Secure data handling

## Purpose
This project was developed as part of academic learning to understand data mining applications in healthcare.
